# IMDCS (Intelligent Medical Diabetes Care System) 🩺🤖

نظام ذكي متكامل مدعوم بوكلاء الذكاء الاصطناعي (AI Agents) لمراقبة وإدارة مستويات سكر الدم لمرضى السكري في الوقت الفعلي.

## ✨ الميزات الرئيسية
- **حساس افتراضي مدمج (Virtual CGM):** محاكاة واقعية لقراءات السكر لـ 5 أنماط مختلفة من المرضى.
- **وكلاء ذكاء اصطناعي (LangGraph):** شبكة من الوكلاء يحللون البيانات، يتنبؤون بالاتجاهات (باستخدام خوارزمية EWMA + Momentum)، ويقدمون توصيات طبية دقيقة.
- **توصيات طبية مخصصة:** 7 سيناريوهات طبية (من الهبوط الحرج إلى الارتفاع الحرج) تقدم خطوات علاجية، نصائح وقائية، واقتراحات للوجبات باللغتين العربية والإنجليزية.
- **واجهة مستخدم تفاعلية (React):** لوحة تحكم Premium بتصميم زجاجي (Glassmorphism) تعرض المنحنيات والنصائح بشكل لحظي عبر الـ WebSocket.
- **واجهة خلفية قوية (FastAPI):** تدير التزامن، وتوفر REST APIs لبدء/إيقاف الحساسات وحقن الأحداث السلوكية (مثل تناول وجبة أو أخذ إنسولين).

## 🚀 التقنيات المستخدمة
- **الخلفية (Backend):** Python, FastAPI, LangGraph, WebSockets.
- **الواجهة الأمامية (Frontend):** React, TypeScript, Vite, Recharts, CSS.

## 🛠️ كيفية التشغيل

### 1️⃣ تشغيل السيرفر (الخلفية)
```bash
# تأكد من أنك في المجلد الجذري للمشروع
python -m backend.main
```
*سيعمل السيرفر على `http://localhost:8000` وستبدأ الحساسات الـ 5 بالعمل تلقائياً.*

### 2️⃣ تشغيل الواجهة (الأمامية)
افتح نافذة Terminal جديدة واكتب:
```bash
cd frontend
npm install
npm run dev
```
*افتح الرابط الذي سيظهر في المتصفح (غالباً `http://localhost:5173`) لمشاهدة لوحة التحكم الحية.*

## 📂 هيكلية المشروع
- `/backend`: يحتوي على السيرفر (FastAPI)، وكلاء الذكاء الاصطناعي، ومحرك التوصيات.
- `/frontend`: يحتوي على واجهة React التفاعلية والمكونات المرئية.
- `/simulator`: يحتوي على ملفات تعريف المرضى (Patient Profiles) ومولد القراءات الافتراضية.
